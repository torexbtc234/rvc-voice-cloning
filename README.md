# RVC Voice Cloning System for RunPod Serverless

Real-time Retrieval-based Voice Conversion (RVC) system with WebSocket streaming for ultra-low latency voice conversion.

## Features

- 🎤 **Voice Training**: Upload 20-30 second audio sample to clone any voice
- ⚡ **Real-time Conversion**: WebSocket streaming with <100ms latency
- 📱 **Mobile Responsive**: Works on all modern browsers
- 🚀 **GPU Accelerated**: Optimized for RunPod GPU instances
- 🔒 **Secure**: API key authentication for HTTP and WebSocket endpoints

## Prerequisites

1. **RunPod Account** with credits
2. **GitHub Account** (for pushing code)
3. **Docker Hub Account** (optional, for custom image)
